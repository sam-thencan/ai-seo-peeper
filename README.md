A Chrome Extension that reveals the hidden "fan-out queries" ChatGPT uses when browsing the web. 

## How to Install
1. Download this repository (Code -> Download ZIP) and unzip it.
2. Open Chrome and go to `chrome://extensions/`.
3. Toggle **Developer mode** (top right).
4. Click **Load unpacked**.
5. Select the unzipped folder.

## How to Use
1. Go to ChatGPT.
2. Ask a question that requires browsing (e.g., "Find me 3 pizza places in Chicago").
3. Watch the "Peeper" box appear in the top right with the exact search terms used!
